## 💰 定价策略分析

### Typeless
- **免费版**: 4,000 词/周
- **Pro**: $12/月 (年付), $30/月 (月付)
- **特点**: 限制字数，推动付费

### Spokenly
- **免费**: 本地模型无限使用 (Whisper/Parakeet)
- **BYOK**: 免费使用自己的 API Key
- **Pro**: $7.99/月 (云端模型)
- **特点**: 本地免费，云端付费

### SuperWhisper
- **免费**: 基础模型
- **Pro**: $8.49/月, $84.99/年
- **终身**: $249.99
- **特点**: 提供终身买断选项

### AitoType 定价建议
```
推荐采用 Spokenly 模式：
- 免费: 支持 BYOK，用户自己的 API Key 无限使用
- Pro: 内置 API 配额（如果未来考虑）

优势：
1. 用户用自己的 Key，成本极低（Groq 免费、OpenAI 便宜）
2. 降低用户付费门槛
3. 避免服务器成本
```

## 🌟 AitoType 差异化定位

基于竞品分析，AitoType 的核心定位应该是：

### 核心卖点
1. **BYOK (Bring Your Own Key)** - 完全免费使用自己的 API
2. **多模型支持** - OpenAI、Groq、阿里云、豆包一键切换
3. **开源/透明** - 代码可审计，隐私有保障
4. **轻量快速** - Tauri 打包，体积小、启动快

### 与竞品的区别

| vs Typeless | vs Spokenly | vs SuperWhisper |
|-------------|-------------|-----------------|
| 支持 BYOK，无字数限制 | 支持更多模型（阿里/豆包） | 更轻量、更简单 |
| 支持离线模型 (Phase 2) | 开源可审计 | 价格更低/免费 |
| 支持 Windows (Phase 2) | 支持 Windows (Phase 2) | 无复杂配置 |

## 🎨 UI/UX 参考

### Typeless 设计亮点
- 极简的录音状态悬浮窗
- 打字速度对比可视化 (45 wpm vs 220 wpm)
- 清晰的隐私承诺展示

### Spokenly 设计亮点
- "Talk, don't type" 的清晰价值主张
- 功能分类清晰 (Languages / Privacy / Agent Mode / AI-Powered)
- 透明的定价表格

### 建议 AitoType 采用
1. **极简悬浮窗**: 录音时显示动态波形，不干扰用户
2. **托盘菜单**: 快速切换模型、查看状态
3. **设置界面**: 模块化，API 配置 / 快捷键 / 高级选项分开
