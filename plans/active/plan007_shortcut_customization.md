# plan007 - 快捷键冲突提醒与组合录制增强

- 创建时间: 2026-02-13
- 状态: 已完成
- 目标: 在快捷键设置中增加冲突提醒，并支持 1~3 键自由组合（不强制功能键）。

## 问题描述
当前快捷键录制逻辑仅支持“修饰键 + 单个按键”的形式。
当用户设置与当前快捷键相同的值时，也不会给出明确提醒。

## 执行计划
1. 梳理现有快捷键录制与保存流程，确认可插入提醒与校验的位置。
2. 调整录制逻辑，支持 1 键、2 键、3 键组合（含非功能键组合）。
3. 增加“与当前快捷键相同”提醒，避免无效重复设置。
4. 验证快捷键设置流程与基础构建是否正常。
5. 提交修复。

## 执行结果
- 已在独立 worktree `worktrees/shortcut-customize`（分支 `codex/shortcut-customize`）完成修复，避免影响主工作区未提交改动。
- 快捷键录制改为支持 1~3 键组合，不再限制为“修饰键 + 单键”，支持单键、双键、三键组合。
- 新增重复设置提醒：当新录制快捷键与当前快捷键一致时，显示“无需重复设置”的警告提示。
- 新增快捷键提示区，展示录制引导、成功、警告和失败信息。
- 已完成语法校验：`node --check src/aitotype/src/main.js`。
